What I've Done

A collection of demos, tools, and interactive experiences I've built

Industry Agents

An interactive character selection interface showcasing industry-specific AI assistants. When a character is selected, a video is played to give a short preview of how CM.com's HALO can automate those kinds of experiences through Agentic AI and Tools. I built this project so that it could be used at a trade show on a touchscreen monitor (so people passing by had a fun way to learn more about what we can do without sitting through a full demo).

CM.com North America Halo Demo Hub

This is the CM.com North America Halo Demo Hub, where team members can showcase fully customized demos of CM.com's HALO solution. Each user has a dedicated space to create personalized chatbot demos by uploading a screenshot of a business's webpage and tailoring the HALO chatbot experience to that customer. I built this project so that all team members had a dedicated place to explore and test the capabilities of HALO, while also giving them a space to demo the product in a tailored way.

BitCRM

This is a mock-up of a CRM platform meant to mimic a potential client's own CRM solution. This pulls in data from a DummyJSON API call (the client was using PearlDiver to collect data, so this is a way to mimic that). From there, I built out the option to push the CRM profiles over to CM.com's Customer Data Platform (CDP) so that we can demonstrate how automation, workflows, and Email Campaigns can be triggered based on the properties of the CRM entry.
